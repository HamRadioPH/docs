How to collaborate

Ham Philippines wiki is hosted on Github, this platform is incredibly effective way to collaborate between contributors. Providing a place for everyone with an internet connection to have an avenue where they can share information with the world for free. Github has been adopted by many large open-source projects as their primary home for collaboration and contribution.

Currently wiki has this following categories:

  • Course a curated course for amateur radioe xam.
  • Tutorials a collection of amateur related tutorials, guides, primers and how-tos.
  • Projects a collection of diy projects that you want to published, this can be also use as proof of amateur activity.
  • Policies a collection of official documents from NTC, PARA and IARU.

How to get started?

Recommended setup (direct)
Warning some instruction below are technical. Nerd alert!

To get started with collaboration first you need to:

  1. Register an account on Github
  2. Once you have an account you can then install Github Desktop App just download according to your operating system.
  3. Clone our wiki respository https://github.com/HamRadioPH/wiki.git, make sure you select the latest branch version.
  4. Thats it, you can now start editing and collaborating, once you have made some changes commit and push the changes so we can then check and merge it with the master version to be published.
The easy way (slow)

If the instructions above is technical for you just send your edit to wiki@hamradioph.com or drop us a message in any of your social media account, we'll try to respond to it as soon as possible.

Publishing

Once your content gets approved and transfered to master version it will then immediately published to wiki.hamradioph.com.
